# CPM-FM Release 2.27.1 Summary

This release (v2.27.1) is a known good version of the CP/M File Manager. It contains all core features and stable functionality before the major changes to disk image handling introduced in version 2.36.0.

## Key Features in 2.27.1:
- **X-Modem Transfers:** Reliable single/multi-file transfers between host and CP/M systems.
- **Drag and Drop:** Easy file movement between panes and from the host OS.
- **File Conflict Resolution:** Prompted options to Overwrite, Skip, or Cancel existing files.
- **CP/M 8.3 Filename Validation:** Ensures compatibility with legacy filename constraints on upload.
- **Transfer History:** Persistent record of all transfer attempts (success, failure, etc.).
- **Whole-drive Backup and Restore:** Mirror entire remote drives to the host or vice versa.
- **VT-100 Terminal:** Fully functional terminal window with escape sequence support.
- **Configurable Macros:** Support for up to ten programmable keystroke macros.
- **Multi-language UI:** Support for 12 languages including English, Spanish, French, German, etc.
- **Material Design Theme:** Modern UI following host OS light/dark modes.

**Note:** Use this version if your workflow relies on the pre-2.36.0 disk image management behavior.

Any newer versions than 2.27.1 listed here are for Windows users who want to use the app without having to build it. Note: The newer package may not be from the latest updates to main.
